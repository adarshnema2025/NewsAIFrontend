Build a responsive frontend using **React.js + Tailwind CSS** for a News AI application.

## 🔧 Tech Requirements

* React.js means jsx not tsx(functional components + hooks)
* Tailwind CSS for styling
* React Router for navigation
* Axios for API calls
* Context API for global state (auth + user data)

---

## 🔐 AUTHENTICATION FLOW

### Pages:

1. Login Page
2. Signup Page

### Features:

* JWT-based authentication
* OAuth2 (Google login button UI only if backend not ready)
* Store token in localStorage
* Redirect to homepage after login
* Protected routes

---

## 🏠 MAIN HOME PAGE (After Login)

### 1. Navbar

* Logo: "NewsAI"
* Categories (clickable): Business, Tech, Sports, Health, etc.
* Profile Avatar (right side)

  * Dropdown:

    * Dashboard
    * Archives
    * Logout

---

### 2. News Feed Section

Display news as **responsive cards grid**

#### Each Card contains:

* Image (top)
* Title
* Source/Author
* 3 Buttons:

  1. **AI Summary**

     * Opens modal with summary (API call)
  2. **Read Original**

     * Opens article URL in new tab
  3. **Add to Notebook**

     * Dropdown:

       * List of user notebooks
       * "Create New Notebook" option

---

### 🧠 Add to Notebook Flow

* On click → dropdown opens
* Select notebook → API call
* OR
* Click "Create New Notebook"

  * Opens modal
  * Input: name + description
  * Create → auto-select

---

## 📊 DASHBOARD PAGE

#### Each Notebook Card:
